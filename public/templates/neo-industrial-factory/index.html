Tolerance Specifications

What sensor hardware is required? +

AXIS integrates with existing industrial sensor networks (Modbus, OPC UA, BACnet). For greenfield deployments, we provide certified sensor kits rated for -40°C to +85°C operation with 10-year MTBF. All hardware carries ISO 13849 PLd certification.

How is prediction accuracy validated? +

All predictions are back-tested against 4.2PB of historical failure data from 1,800+ industrial facilities. Accuracy metrics are continuously validated against ground-truth failure events with third-party auditors. Current field-validated accuracy: 99.94% for mechanical systems, 99.87% for electrical infrastructure.

What happens during network partition? +

Edge nodes operate autonomously during network loss with 72 hours of local storage and full prediction capability. All critical alerts trigger local SMS/cellular fallback. Upon reconnection, edge nodes synchronize state with core without data loss. Zero single point of failure in alerting path.
