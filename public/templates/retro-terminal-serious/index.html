## Core differentiators
 
--no-gimmicks
Every CLI metaphor maps to real product function. No decorative elements that break terminal authenticity.
--phosphor-glow
CRT text-shadow applied only to primary conversion elements (metrics, CTAs). Secondary text remains dimmed for hierarchy.
--scanline-overlay
Ambient motion via fixed scanline (3% opacity) — creates depth without breaking terminal metaphor or causing distraction.
--cursor-persistence
Blinking cursor always present on active element — signals "live system" not static marketing page.
--zero-radius
Absolute prohibition on border-radius. All elements obey terminal geometry: sharp edges only.
--monospace-purity
100% monospace throughout — no exceptions for body copy, legal text, or metadata. One typeface rules all.
